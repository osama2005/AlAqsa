# نظام إدارة المخزون - مستشفى شهداء الأقصى

نظام يعمل بالكامل على المتصفح (Client-side) لإدارة المخزون والمستودعات.

- **لا يوجد سيرفر ولا قاعدة بيانات** — جميع البيانات تُحفظ في `localStorage`
- واجهة باللغة العربية (RTL)
- خط Amiri في جميع الصفحات
- دعم الوضع الليلي (Dark Mode)
- دعم تغيير خلفية الموقع (صور كواكب)
- متجاوب مع جميع أحجام الشاشات (360px → 1400px)

## صفحات النظام

### 1. صفحة تسجيل الدخول (`index.html`)

| العنصر | الوصف |
|--------|-------|
| **اسم المستخدم** | حقل إدخال مخفي (`type="password"`) مع زر 👁️ للإظهار/الإخفاء |
| **زر الوضع الليلي** | يبدل بين الوضع الفاتح والداكن (أعلى اليسار) |
| **الساعة الحية** | تعرض الوقت والتاريخ (أعلى اليمين) |
| **تغيير الخلفية** | قائمة منسدلة لاختيار صورة كوكب |
| **الدعم الفني** | رابط بريد إلكتروني (`mailto:osamaandanes@gmail.com`) |
| **دخول** | زر تسجيل الدخول (يتحقق من وجود المستخدم) |

- لا توجد كلمة مرور — النظام يعتمد على اسم المستخدم فقط
- المستخدم الافتراضي: `admin123` (دور: admin، صلاحية تعديل)
- يتم إنشاء المستخدم الافتراضي تلقائياً عند أول تشغيل

### 2. لوحة تحكم المدير (`admin/index.html`)

إدارة المستخدمين والصلاحيات.

| الحقل | الوصف |
|-------|-------|
| **اسم المستخدم** | المعرف الفريد لكل مستخدم |
| **الدور** | `admin` (أدمن) أو `user` (مستخدم عادي) — الأدمن يرى لوحة التحكم، المستخدم يرى المخزون فقط |
| **صلاحية التعديل** | نعم (يمكنه إضافة/تعديل/حذف) أو لا (مشاهدة فقط) |
| **إجراءات** | تعديل / حذف |

**المستخدمون النشطون:** نظام Heartbeat — كل مستخدم يُحدّث وقته النشط كل 15 ثانية، يعتبر نشطاً إذا كان آخر نشاط له أقل من 60 ثانية.

### 3. صفحة المخزون (`inventory/index.html`)

تنقسم إلى 5 تبويبات على نمط Headless UI:

#### 3.1. تبويب المخزون (Inventory)

جدول الأرصدة والبيانات الأساسية.

| الحقل | الوصف |
|-------|-------|
| **كود الصنف** | رقم فريد لكل صنف — يُنشأ تلقائياً |
| **كود وزارة** | كود وزارة الصحة (اختياري) |
| **اسم الصنف** | اسم المادة أو الجهاز |
| **الوحدة** | وحدة القياس (عدد، كرتون، لتر، ...) |
| **التصنيف** | أجهزة طبية / كهربائية / أقمشة / أثاث / مستلزمات طبية / أدوية / مواد تنظيف / قرطاسية / أخرى |
| **النوع** | مستهلك أو مستديم |
| **رصيد أول المدة** | الرصيد الابتدائي |
| **الوارد** | إجمالي الكميات الواردة (يُحتسب تلقائياً) |
| **الصادر** | إجمالي الكميات المنصرفة (يُحتسب تلقائياً) |
| **الرصيد المتبقي** | openingBalance + totalIn - totalOut |
| **السعر** | سعر الوحدة |
| **ملاحظات** | أي ملاحظات إضافية |

#### 3.2. تبويب الوارد (Incoming)

تسجيل البضائع المستلمة (إضافة للمخزون).

| الحقل | الوصف |
|-------|-------|
| **اليوم / التاريخ** | يُعبآن تلقائياً |
| **كود الصنف** | اختيار الصنف من قائمة منسدلة (يتم تعبئة باقي الحقول تلقائياً) |
| **الكمية** | الكمية الواردة |
| **الجهة الوارد منها** | نص حر — يكتبه المستخدم |
| **السعر / الإجمالي** | الإجمالي = qty × price (يُحتسب تلقائياً) |

#### 3.3. تبويب الصادر (Outgoing)

تسجيل البضائع المصروفة للأقسام (خصم من المخزون).

- يتم التحقق من أن الكمية لا تتجاوز الرصيد المتاح
- **الجهة المصروف لها:** اسم القسم المستلم

#### 3.4. تبويب الإعارة (Loan)

إدارة المعدات الطبية المعارة.

- **قسم 1 — جدول الإعارة (الأرصدة):** كود، اسم، وحدة، تصنيف، رصيد أول المدة، وارد، صادر، متبقي
- **قسم 2 — سجل الإعارة الوارد:** اليوم، التاريخ، كود، اسم، كمية، جهة، سعر، إجمالي

#### 3.5. تبويب الكهنة (Kahana)

سجل بسيط للكهنة (أسماء وتفاصيل).

## نظام التخزين (localStorage Keys)

| المفتاح | المحتوى |
|---------|---------|
| `inv_users` | مصفوفة المستخدمين |
| `inv_current_user` | المستخدم الحالي |
| `inv_auth` | حالة المصادقة (`'active'` أو `null`) |
| `inv_sessions` | جلسات المستخدمين النشطون |
| `inv_theme` | `'dark'` أو `'light'` |
| `inv_bg` | اسم الخلفية |
| `inv_items` | مصفوفة أصناف المخزون |
| `inv_incoming` | مصفوفة سجلات الوارد |
| `inv_outgoing` | مصفوفة سجلات الصادر |
| `inv_loan` | مصفوفة أصناف الإعارة |
| `inv_loan_incoming` | مصفوفة سجلات وارد الإعارة |
| `inv_kahana` | مصفوفة الكهنة |
| `_colColors` | ألوان الأعمدة المخصصة |

## نظام CSS والثيمات

- **الخط:** Amiri (Google Fonts) + Tahoma + Arial fallback
- **الأيقونات:** Font Awesome 6.5.1 (CDN)
- **الوضع الفاتح:** خلفية بيج فاتح (#f5efe0)، حاويات بيضاء (#fff)
- **الوضع الداكن:** خلفية كحلي غامق (#1a1a2e)، حاويات #1e2a3a، ذهبي (#c8a24e)
- **تأثيرات:** fadeIn، slideUp، shake، pulse، themeFlash
- **استجابة:** >1024px كامل، 768px متوسط، 480px ضيق، 360px أدنى

## نظام تخصيص الألوان المتقدم

- **لون خلفية الصف** — يغير خلفية الصف بأكمله
- **لون نص الصف** — يغير لون النص في الصف بأكمله
- **لون العمود** — يغير لون النص في عمود معين (كل الصفوف)
- **لون الخلية** — يغير لون النص في خلية محددة (صف + عمود)

يمكن الوصول عبر زر 🎨 بجانب كل صف أو عبر زر "تخصيص الألوان" في شريط الأدوات.

## كيفية الاستخدام (Quick Start)

1. افتح `index.html` في المتصفح
2. سجل دخول بـ: `admin123`
3. ستنتقل إلى لوحة التحكم (`admin/index.html`)
4. أضف مستخدمين جدد (يظهرون في صفحة الدخول)
5. اذهب للمخزون عبر زر "المخزون"
6. أضف أصنافاً جديدة في تبويب "المخزون"
7. سجل وارد / صادر في التبويبات المخصصة
8. استخدم الإعارة والكهنة حسب الحاجة
9. صدر البيانات إلى Excel عند الرغبة

## ملاحظات فنية

- جميع البيانات في `localStorage` — لا تفقد البيانات بتحديث الصفحة
- يمكن مسح `localStorage` من أدوات المطور (F12 ← Application)
- يوجد مستخدم افتراضي `admin123` في أول تشغيل
- لا توجد كلمة مرور — النظام مبسط للاستخدام الداخلي
- تصدير Excel يستخدم مكتبة SheetJS (CDN)
- الأصوات عبر Web Audio API — لا تحتاج ملفات صوتية
- التوافق: جميع المتصفحات الحديثة (Chrome, Firefox, Edge, Safari)
