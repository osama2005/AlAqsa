## كيفية تشغيل الموقع

1. افتح ملف `index.html` في أي متصفح حديث (Chrome, Firefox, Edge)
2. ستظهر صفحة تسجيل الدخول
3. أدخل بيانات المستخدم الافتراضي (أو أي مستخدم تم إنشاؤه)
4. سيتم توجيهك تلقائياً إلى لوحة المخزون أو لوحة التحكم حسب صلاحياتك

> **ملاحظة:** لا حاجة لسيرفر محلي أو اتصال بالإنترنت (باستثناء خط Font Awesome و SheetJS من CDN).

## نظام الصلاحيات

نوعان من المستخدمين:

### 1. مدير (admin)
- `role: "admin"`
- `canEdit: true`
- يستطيع تعديل وحذف الأصناف
- يدخل إلى المخزون ولوحة التحكم
- يستطيع إدارة المستخدمين (إضافة، تعديل، حذف)
- يظهر له زر الذهاب إلى لوحة التحكم في القائمة الجانبية للمخزون

### 2. مستخدم عادي (user)
- `role: "user"`
- `canEdit: false` (مشاهدة فقط)
- لا يستطيع إضافة أو تعديل أو حذف الأصناف
- لا يستطيع الوصول إلى لوحة التحكم
- جميع أزرار التعديل والحذف مخفية

### التوجيه عند تسجيل الدخول

بعد تسجيل الدخول:
- **admin** → يذهب إلى `inventory/index.html` (المخزون)
- **user** → يذهب إلى `inventory/index.html` (المخزون)
- أي مستخدم يحاول الدخول إلى `admin/index.html` وليس لديه صلاحية admin يتم توجيهه إلى `index.html`

## الصفحات

### 1. تسجيل الدخول (`index.html`)

- تصميم glassmorphism (زجاجي) مع خلفية نجوم تفاعلية (starfield parallax)
- حقل اسم المستخدم + كلمة مرور مع زر إظهار/إخفاء
- تأثير lamp overlay عند تمرير الماوس
- صوت ترحيبي عند تسجيل الدخول
- التحقق من اسم المستخدم وكلمة المرور من `localStorage`
- توجيه تلقائي حسب الدور
- دعم الوضع الليلي فوراً

### 2. المخزون (`inventory/index.html`)

الرئيسية — لوحة تحكم المخزون مع تبويبات فرعية:

| التبويب | الوصف |
|---------|-------|
| المخزون (الأصناف) | عرض وإضافة وتعديل وحذف الأصناف مع إحصائيات |
| الوارد | تسجيل الواردات مع الربط بالأصناف |
| الصادر | تسجيل الصادرات مع التحقق من الرصيد |
| الإعارة | إدارة الأصناف المعارة |
| التقارير | إحصائيات ورسوم بيانية |
| خيمة الكحانة | إدارة محتويات خيمة الكحانة |

### 3. لوحة التحكم (`admin/index.html`)

- إدارة المستخدمين (إضافة، تعديل، حذف)
- عرض المستخدمين في جدول مع كلمة المرور (معروضة بنقاط)
- إحصائيات: إجمالي المستخدمين، المسجلين حالياً، الجلسات النشطة
- مراقبة الجلسات النشطة مع آخر نشاط لكل مستخدم
- زر العودة إلى المخزون

## شرح الأقسام بالتفصيل

### المخزون (الأصناف)

- جدول يعرض جميع الأصناف المسجلة
- كل صنف يحتوي على: كود، كود وزارة، اسم، وحدة، تصنيف، نوع، رصيد أول، وارد، صادر، متبقي، سعر، ملاحظات
- إحصائيات في أعلى الصفحة: إجمالي الأصناف، إجمالي الكمية، القيمة الإجمالية
- أزرار: إضافة صنف، تحديث الإحصائيات، تصدير Excel
- كل صف له زر تخصيص لون (heroUI color picker)
- أزرار تعديل وحذف (تظهر فقط للمستخدمين الذين لديهم `canEdit: true`)

#### إضافة صنف جديد

- كود الصنف يُنشأ تلقائياً (تسلسلي)
- **الوحدة** — قائمة منسدلة ديناميكية مع زر `+` لإضافة وحدة جديدة
- **التصنيف** — قائمة منسدلة ديناميكية مع زر `+` لإضافة تصنيف جديد
- **النوع** — قائمة منسدلة ديناميكية مع زر `+` لإضافة نوع جديد
- رصيد أول المدة، الوارد، الصادر — مع حساب آلي للرصيد المتبقي

### الوارد

- تسجيل الواردات وربطها بالأصناف
- عند اختيار كود الصنف، يتم تعبئة باقي الحقول تلقائياً (الاسم، الوحدة، التصنيف، النوع)
- إجمالي الوارد = الكمية × السعر
- بعد الحفظ، يتم تحديث رصيد الصنف تلقائياً

### الصادر

- تسجيل الصادرات مع التحقق من كفاية الرصيد
- عند اختيار كود الصنف، يتم تعبئة باقي الحقول تلقائياً
- تحذير إذا كانت الكمية المنصرفة أكبر من الرصيد المتاح
- بعد الحفظ، يتم تحديث رصيد الصنف تلقائياً

### الإعارة

- إدارة الأصناف المعارة بشكل منفصل
- ربط بالإصناف الموجودة في المخزون
- التصنيف (قائمة منسدلة ديناميكية)

### التقارير

- إحصائيات عامة
- رسوم بيانية (charts)
- تقارير الوارد والصادر

### خيمة الكحانة

- إدارة مخصصة لمحتويات خيمة الكحانة
- اسم، تصنيف، تفاصيل، ملاحظات

### المستخدمون (لوحة التحكم)

- إضافة مستخدم جديد: اسم مستخدم، كلمة مرور، دور (admin/user)، صلاحية تعديل
- تعديل المستخدم: تغيير كلمة المرور، الدور، صلاحية التعديل
- حذف المستخدم
- عرض كلمة المرور بنقاط (●●●●)
- إحصائيات المستخدمين والجلسات النشطة

## المميزات

### القائمة الجانبية (Sidebar)

- تحتوي على روابط الأقسام المختلفة
- في الديسكتوب:
  - العرض الافتراضي 250px مع النصوص
  - يمكن طيها إلى 65px (أيقونات فقط) عبر زر "طي القائمة" في الأسفل
  - عند الطي: تختفي النصوص وتظهر الأيقونات فقط
- في الجوال (< 1024px):
  - مخفية بشكل افتراضي
  - تظهر عند الضغط على زر القائمة (hamburger menu) في الأعلى
  - تغطية الشاشة بالكامل مع خلفية شفافة (overlay)
  - إغلاق بالضغط على overlay أو زر القائمة

### أزرار النيون

جميع الأزرار بتصميم خاص:
- شكل دائري (pill-shaped) مع border-radius: 50px
- خط متوهج في الأعلى والأسفل عند hover (::before / ::after)
- ظل متوهج (box-shadow) عند hover
- ألوان مخصصة: أخضر #10b981 للحفظ، أحمر للتراجع، أزرق للتعديل

### شريط تحميل 8bit

- يظهر عند التنقل بين الصفحات (المخزون ← لوحة التحكم والعكس)
- 20 بلوك تتعبأ تدريجياً (كل 75ms)
- إطار مزدوج (retro border effect)
- نسبة مئوية تظهر فوق الشريط

### لوحة تخصيص الألوان (heroUI)

- لكل صنف يمكن تخصيص لون خلفية مخصص
- 12 لوناً متاحاً في شبكة (color picker)
- عند الضغط على زر اللون، تظهر اللوحة فوق الزر مباشرة
- إغلاق اللوحة بالضغط على اللون المختار أو خارجها
- دعم الوضع الليلي

### الحقول الديناميكية

الوحدة، التصنيف، والنوع — جميعها:
- قوائم منسدلة (<select>) مسحوبة من `localStorage`
- زر `+` بجانب كل حقل لإضافة قيمة جديدة
- عند إضافة قيمة جديدة، تُحفظ في `localStorage` وتظهر فوراً في القائمة
- يتم اختيار القيمة الجديدة تلقائياً بعد الإضافة
- فرز أبجدي للقيم
- منع إضافة قيم مكررة
- في التعديل: إذا كان للصنف قيمة غير موجودة في القائمة، تُضاف تلقائياً
- في أول تحميل: تُستخرج القيم من الأصناف الموجودة (seeding)

### دعم الخط Marhey

- الموقع كامل باللغة العربية (RTL)
- خط Marhey من Google Fonts
- أحجام خطوط مناسبة لكل العناصر

### التصميم المتجاوب (Responsive)

- ديسكتوب (> 1024px): عرض كامل مع شريط جانبي 250px
- تابلت (768px - 1024px): شريط جانبي منزلق، قائمة طعام مخفية
- جوال (< 768px): كل شيء مصغر، جداول بتمرير أفقي، شبكة إحصائيات بعمودين
- جوال صغير (< 480px): إخفاء الساعة، أزرار أصغر، هوامش صغيرة

## الحقول الديناميكية

### كيف تعمل

1. عند فتح مودال الإضافة أو التعديل، تُقرأ القيم من `localStorage`
2. تُعرض في `<select>` مع خيار "اختر القيمة" في البداية
3. زر `+` بجانب كل حقل يستدعي `addFieldItem(key, selectId, placeholder)`
4. `addFieldItem`:
   - يطلب من المستخدم إدخال القيمة الجديدة عبر `prompt()`
   - يتحقق من عدم التكرار
   - يضيف القيمة إلى المصفوفة، يفرزها، ويحفظها
   - يحدّث `<select>` ويعيّن القيمة الجديدة كمختارة

### أين تظهر

- إضافة/تعديل صنف: الوحدة، التصنيف، النوع
- إضافة/تعديل إعارة: التصنيف فقط

## الوضع الليلي

- زر دائري في الشريط العلوي (topbar)
- SVG أيقونة مع mask: شمس ← قمر morph
- انتقال (transition) مع cubic-bezier spring
- الألوان تتغير:
  - الخلفية: أبيض ← رمادي غامق
  - البطاقات: أبيض ← أسود
  - النصوص: أسود ← أبيض
  - الأزرار: حواف تتكيف مع الوضع الليلي
    يتم حفظ الحالة في `localStorage` (مفتاح `inv_theme`)

## دليل المطور

### إضافة زر نيون جديد

```html
<button class="btn-action">نص الزر</button>
```

متوفر كلاسات:
- `btn-action` — زر عادي
- `save-btn` — زر حفظ (أخضر)
- `cancel-btn` — زر إلغاء (أحمر)

### إضافة حقل ديناميكي جديد

```js
// تعريف حقل جديد بقائمة ديناميكية
fieldSelectHtml('inv_mykey', 'my_field_id', selectedValue, 'اختر القيمة')

// إضافة قيمة جديدة
addFieldItem('inv_mykey', 'my_field_id', 'اختر القيمة')
```

### تدفق البيانات

1. المستخدم يضيف صنفاً ← `saveInventoryItem()` ← `inventory.push()` ← `save()`
2. `save()` تخزن كل المصفوفات في `localStorage`
3. `renderAll()` تعيد رسم جميع الجداول
4. عند الوارد/الصادر: `recalcInventoryTotals(code)` تحدث رصيد الصنف

### إضافة وحدة قياس جديدة (مثال)

```js
// في الكونسول:
addFieldItem('inv_units', 'item_unit', 'اختر الوحدة');
// أو من خلال واجهة المستخدم: الضغط على زر + في حقل الوحدة
```

### أنظمة الألوان

المتغيرات الأساسية في `shared.css`:
```css

--primary-hover: #0d5e36;
--sidebar-bg: #142918;
--neon-color: #10b981;
```

لون القائمة الجانبية: `#142918` (أخضر غامق)
لون النيون: `#10b981` (أخضر فاتح)
